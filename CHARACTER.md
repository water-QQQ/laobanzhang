## TTS（对接本仓库）

- **`tts_voice_id`**：（可留空）填表 `aigc_tts.voice_id` 同含义的供应商音色串；调用 `POST {BASE}/api/aigc/tts/generate` 时放入 JSON 的 **`voice_id`**。未填则省略该字段（默认音色）。
- **`{BASE}`** 示例：`https://agnes-test-gcp.kiwiar.com/`
- 取音频：`data.generate_result.data.uri`（且 `data.generate_result.code == 0`）。

试调用示例：

```bash
curl -sS -X POST "{BASE}/api/aigc/tts/generate" \
  -H "Content-Type: application/json" \
  -d "{\"text\": \"听着，先把事儿办了，别整那些花里胡哨的。\",\"voice_id\": \"你的tts_voice_id\",\"run_id\": \"00000000-0000-0000-0000-000000000001\"}"
```
